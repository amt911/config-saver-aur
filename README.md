# config-saver-aur

`config-saver` is a simple utility designed to help you back up and restore your configuration files on Arch Linux systems. This package is intended for local installation, so you need to clone this repository from GitHub and build it manually.

## Features

- Save and restore configuration files easily
- Compatible with Arch Linux and derivatives
- Lightweight and easy to use

## Installation

To install `config-saver` locally, clone this repository and build the package manually:

```sh
git clone https://github.com/amt911/config-saver-aur.git
cd config-saver-aur
makepkg -si
```
